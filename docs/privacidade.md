# Privacidade

Este repositório não contém:

- código-fonte do sistema real;
- dados fiscais;
- CNPJs reais;
- nomes de clientes ou fornecedores reais;
- chaves NFS-e;
- planilhas, PDFs, XMLs ou bancos de dados;
- instaladores ou executáveis.

As imagens são demonstrativas e utilizam informações fictícias.
